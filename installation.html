Jenkins Installation Guide

Set up Jenkins on Linux, Windows, or Docker. After setup, unlock Jenkins, create admin user, and install recommended plugins.

Linux (Ubuntu/Debian)

sudo apt update
sudo apt install -y fontconfig openjdk-17-jre
curl -fsSL https://pkg.jenkins.io/debian-stable/jenkins.io-2023.key | sudo tee /usr/share/keyrings/jenkins-keyring.asc > /dev/null
echo deb [signed-by=/usr/share/keyrings/jenkins-keyring.asc] https://pkg.jenkins.io/debian-stable binary/ | sudo tee /etc/apt/sources.list.d/jenkins.list > /dev/null
sudo apt update && sudo apt install -y jenkins
sudo systemctl enable --now jenkins

Windows

  • Install Java 17 (Temurin/OpenJDK).
  • Download Jenkins MSI installer.
  • Run installer as Administrator.
  • Open http://localhost:8080 and paste initialAdminPassword from Jenkins secrets folder.

Docker Installation

docker run -d --name jenkins \
  -p 8080:8080 -p 50000:50000 \
  -v jenkins_home:/var/jenkins_home \
  jenkins/jenkins:lts-jdk17

Best for quick learning and disposable environments.

Initial Setup Checklist

  • Unlock Jenkins with initial password.
  • Install suggested plugins.
  • Create first admin user.
  • Configure URL, timezone, and email.
  • Install key plugins: Git, Pipeline, Docker, Blue Ocean, Credentials Binding.
